Troubleshooting:
Mechanics:
1) Coffee does not fly in the roasting chamber
1. Check if all parts of the roaster are tightly closed and not leaking air: Roasting
chamber, chaff collector, Doors, Cyclone, Motors.
2. Check the mesh inside the cyclone system
3. Check if you have not accidentally switched to PC mode instead of PLC mode.
4. Check if you have not switched off the main/cyclone motors and their circuit
breakers in the electrical box.
5. Check all seals if there are no damaged seals and insulation is intact
6. Check motor phasing (if the motors run clockwise).
2) Roasting chamber drawer does not extend out of the roasting chamber
1.Check and adjust the 6 bolts of the roasting chamber (to make sure they are level).
2.Check if the plate is inserted in the roaster correctly, without warping.
3.Check the guides if there is no blockage/obstruction.
3) Rattling noise from the machine
1. Check from service mode each roaster motor separately
2. Check if the chaff collector is well fixed and tightened.
3. Check if the blades of the cooling table are well tightened
4 Check if the hopper is well tightened.
4) Long cooling time (Cleaning table, pipes, checking motor)
1. Check if there is no blockage in the cooling table by removing it beforehand.
2. Check if the cooling pipe inside the roaster and afterwards is not clogged.
3. Check if the cooling motor is working and its circuit breaker is on in the
switchboard.
5) Smoke in the roasting chamber
1. Check if the exhaust pipe opening regulator is functional.
2. Check that there are no foreign objects in the roasting chamber or heating element
chamber.
3. Check if the exhaust pipes inside the roaster and afterwards are not blocked.

6)Whistling sounds/weak draught
1. Check the insulation of the roaster if everything is tightly sealed.
2. Check seals if they are intact
3. Check if the door hinges are tightened
4. Check the insulation of all windows.
7) The coffee roasts unevenly
1. Check the operation of the cyclone motor
2. Check the settings of the extractor pipe
3. Check roaster insulation
Electrics:
1) Main Motor no signal
1. Open the electrical box and check the frequency converter:
If the frequency switch is on-check the error code on it and take a photo. The
photo should be sent to your technologist.
If the frequency converter is not switched on, check the motor circuit
breaker.
2. Check if all three phases go to the roaster.
2) Upper/Lower Temperature sensore defective
1. Check if the contacts of your sensors in the control cabinet are well tightened.
2. Check if the polarity is correct
3. Check the integrity of the cables/sensors) if there is no damage or contamination.
4. Check if the sensors are tightly fitted in their places.
3) Loss of temperature rise
1. Check the operation of the modules in the control cabinet
2. Check Heating elements
3. Check if the circuit breaker of the electric heaters is switched on.
4. Check PC/PLC mode
5. Check the insulation of the roaster
4) No operation of motors

1. Check motor frequency controller
2. Check motor's automat
3. Check the relay of the respective motor
4. Check the continuity of the cables
5) Light
1. Check the circuit breaker of the lamp in the control cabinet
2. Check the bulb if there is no overheating/damage
3. Check the lampholder if there is no damage
4. Check cable if well tightened and no damage
6) Connecting the computer
1. Check the settings of additional software
2. Check cable and connector connections
3. Check the port connection(the correct port is under the cooling table)
4. Check the connection to the controller by checking the ping on the command line
7)The automatic circuit breakers in the electrical box are knocked out
1. Check the contacts on the main switcher.
2. Check the amperage power setting on the controller of the dispenser, it should be
set to maximum. This can be done with screwdriver. If this does not help, replace the
dispenser.
