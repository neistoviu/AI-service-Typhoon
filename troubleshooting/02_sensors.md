# 02 — Temperature Sensors (Upper / Lower)

> **Total cases:** 26  
> **Frequency:** ★★★★★  
> **Affected models:** All, especially 5 kg, 2.5 kg, 10 electro, 20 electro, 30 electro

---

## Symptoms

- Temperature jumps on display
- Chaotic jumps (120→500→700°C) = burned temperature module
- Sensor error message on screen
- Zeros displayed on sensors
- Large difference between upper and lower sensor readings

## Diagnostic Algorithm

1. Check for **beans stuck** in sensor opening
2. Check **sensor fixation** — is it coming loose?
3. Check **seals** — air leak causes false readings!
4. Check **contacts** in electrical cabinet
5. For chaotic jumps → probably **burned temperature module**

## ⚠️ Critical Rules

- **Always check seals** when sensor issues arise — air leaks cause false readings
- On **30 electro**: beans fly into upper sensor opening — this is a **design issue** → install protective plate
- **Spare sensors can be defective** — test before shipping to client
- For **chaotic jumps** (100s of degrees variation) → replace temperature module, not just sensor

## Manual References

- Check sensor contacts are tightened in control cabinet
- Verify polarity is correct
- Inspect cable/sensor integrity for damage or contamination
- Ensure sensors are tightly fitted in their places
