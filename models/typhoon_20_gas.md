# Typhoon 20 kg Gas — Known Issues

## Top Issues (by frequency)

1. **VFD** — failure, replacement needed
2. **Temperature** — burner may not cope, coffee too dark
3. **Burner connection / pressure relay** — communication issues

## Model-Specific Issues

- **Burner 10-45 may not cope** — at 201°C for 7 min, coffee comes out very dark (Agtron 52). Possibly coefficient in software needs adjustment.
- **Pressure relay issues** — can cause burner communication loss
- **Profile won't repeat** — software issue under investigation
- **VFD Error E001** — parameter A006 adjustment only partially helps

## ⚠️ Unresolved

- Low temperatures despite normal settings → possible software coefficient issue
- Burner communication loss
- Profile repetition failure
